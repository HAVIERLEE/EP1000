Finally for the assignment, I will have to use arduino and process to interface with my board. On the processing side, I have code that create a triangle "button" on the screen when my mouse hovers over it changes colour from a bright red to a dark red to indicate that it is press. When it is 'pressed' it communicates with the arduino through the serial port and sends the letter "H". If the button is not pressed, it send a "L". On the Arduino side, it reads the serial port. When a 'H' is received it lights up the LED and when an 'L' is received it turns it off.

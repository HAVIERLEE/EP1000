GUIDE TO 3D PRINTING

In this guide, I am going to show you how to turn a 3D drawing in fusion 360 into a STL file and import it into Cura and 3D print it.

  • A Fusion 360 Drawing file
  • Ultimaker Cura

STEP 1 to 4: Fusion 360 -> Cura

❮ ❯

Step 1: Go to the file and go to the Export Tab.

Step 2: Once in the Export page, under Type change the file type into 'stl'.

Step 3: Open the file in Cura

Step 4: For this print, I want to reduce the amount of support on this to reduce the time to print and for a cleaner print. So use the 'E' key to create a 'Support Block'. A support block prevents the Slicer for generating the supports needed for this print.

STEP 5 to 7: Cura

❮ ❯

Step 5: Using the move tab, I will adjust the Support Block to the chin of the Knight piece. Since as mentioned in the Overview, the overhang angle should allow it to print.

Step 6: Once done, you can go to the bottom right hand corner and hit the "slice" button to begin slicing the print into Gcode.

Step 7: You can preview the print to simulate how the printer would print this piece. Once you are happy, you can hit the save to disk to save it on to a disk to import it into a printer.

With the Object sliced and getting the Gcode, you can check my Blog to see my attempts at 3D printing this piece.
